Day 1 Assignment 2

Write The Difference Between var and const with code example.

Results:

The Variables declared using the var can be updated and also re-declared within its scope
While the variable declared using the let can be updated but not re-declared
but the variables declared using const can not we updated and re-declared
This is the main difference between the var, let, and const

For Example:

Example Of var

var car = "BMW";

var car = "Logan";

The var car has firstly "BMW" as declaration after that the var car has re-declared name which is "Logan" and it will print "Logan" in place of "BMW" after declaration.



Example Of let

let car = "BMW";

let car = "Logan";

In the example above the after re-declaration. if we try to print the car variable it will not print because the let can only update the variable not re- declare it,



Example Of const

const car = "BWM";

const car = "Logan":

In the example above the const variable car will not print beacuse the const variables niether can be updated and nor re-declared



Day 1 Assignment 3

Write a brief intro on available datatypes in Javascript

Results:

There are six basic types of datatypes in Javascript:

1) String
Example: var car = "BMW"; // This is a String which is group of words.

2) Numbers
Example: var car_num = 3; // This is a Number which is a number.

3) Boolean
Example: var car = True ;// This is a Boolean which contains only True and False value

4) Object
Example: var car = { name : "BMW", modal : 4}; // This is a Object which is group of named values

5) Arrays
Example: var car = ["BMW, Logan, Hector"]; // This is a Arrays which can store multiple values in only single variable

6) Null
// A null is nothing, it is supposed to be something that doesn't exist
